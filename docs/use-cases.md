# Real-world use cases

How people actually use 100tools day-to-day.

---

## For developers

### Debugging an API response
1. Paste the raw response into **[JSON Formatter](https://100tools.site/tools/json-formatter)**
2. Compare yesterday's payload vs today's with **[JSON Diff](https://100tools.site/tools/json-diff)**
3. Generate TypeScript types from the cleaned response via **[JSON → TypeScript](https://100tools.site/tools/json-to-typescript)**

### Inspecting an auth token
- Paste a token into **[JWT Decoder](https://100tools.site/tools/jwt-decoder)** to read claims locally — no third-party server sees your token.

### Writing a cron expression
- **[Cron Generator](https://100tools.site/tools/cron-generator)** with human-readable explanations for each field.

### Generating fixture data for tests
- **[Fake Data Generator](https://100tools.site/tools/fake-data-generator)** + **[UUID Generator](https://100tools.site/tools/uuid-generator)** for realistic test seeds.

## For writers and editors

### Polishing a draft
1. **[Word Counter](https://100tools.site/tools/word-counter)** — hit your target length
2. **[Readability Checker](https://100tools.site/tools/text-readability-checker)** — verify reading level
3. **[Remove Line Breaks](https://100tools.site/tools/remove-line-breaks)** — clean up paste artefacts
4. **[Text Diff](https://100tools.site/tools/text-diff)** — review editor changes

### Dedup and sort a list
- Pasted a messy CSV column? **[Text Deduplicator](https://100tools.site/tools/text-deduplicator)** → **[Text Sorter](https://100tools.site/tools/text-sorter)**.

## For designers

### Building a colour system
1. Start with **[Color Palette Generator](https://100tools.site/tools/color-palette-generator)**
2. Validate WCAG contrast in **[Color Contrast Checker](https://100tools.site/tools/color-contrast-checker)**
3. Mock backgrounds with **[CSS Gradient Generator](https://100tools.site/tools/css-gradient-generator)**

### Preparing assets for a website
- Resize and compress in the browser: **[Image Resizer](https://100tools.site/tools/image-resizer)** → **[Image Compressor](https://100tools.site/tools/image-compressor)** → **[Image → WebP](https://100tools.site/tools/image-to-webp)**
- Generate a favicon set from one PNG with **[Favicon Generator](https://100tools.site/tools/favicon-generator)**
