## Tips

- **PNG for screenshots, GIF for demos.** Keep GIFs under 4 MB (lossy compression is fine).
- **Crop tightly** — chrome wastes pixels. Aim for the tool UI to fill ≥ 80 % of the frame.
- **No real user data.** Use lorem-ipsum-style filler.
- **Show one feature per screenshot.** Don't try to capture the whole product in one image.
